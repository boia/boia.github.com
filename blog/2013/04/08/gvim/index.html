初试Vim

因为笔记本没有按ubuntu，所以在windows系统中只好按gvim，我的版本是gvim7.346，如果你也碰到了中文编码问题的话，可以按我说的去解决。



打开_vimrc

打开_vimrc这个文件，这个文件在你安装gvim的根目录可以找到，然后用编译器打开，最好别用记事本。

添加配置

set encoding=utf-8 set fileencodings=utf-8,gbk,gb18030,gk2312 “解决菜单乱码 source $VIMRUNTIME/delmenu.vim source $VIMRUNTIME/menu.vim “解决consle输出乱码 language messages zh_CN.utf-8 “打开语法高亮 syntax on “gvim 配色方案 colorscheme torte

OK，做得好！！！
